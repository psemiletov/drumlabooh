# Drumlabooh: LV2/VSTi drum machine

Drumlabooh is an easy way to write drum tracks as MIDI at your DAW, using Hydrogen, Drumlabooh/Drumrox kits or SFZ.


Drumlabooh uses:
[JUCE](https://juce.com) (GPL3)
