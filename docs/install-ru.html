Windows: установка

Скачайте и запустите установщик VST3 — он поставит плагин и минимальный набор драм-китов.

Дополнительные драмкиты можно взять, например, из драм-машины Hydrogen, а в формате Драмлабуха из Telegram-канала Drum sklad. Распакуйте папки китов в C:\drumlabooh_kits или D:\drumlabooh_kits, а киты формата SFZ — в C:\sfz_kits или D:\sfz_kits.

Подробности — в Руководстве пользователя.

Linux: сборка из исходников

Для сборки Drumlabooh нужны зависимости JUCE (библиотека подгружается автоматически во время сборки).

Зависимости (Ubuntu/Debian)

libasound2-dev
libfreetype6-dev
libx11-dev
libxcomposite-dev
libxcursor-dev
libxext-dev
libxinerama-dev
libxrandr-dev
libxrender-dev
libfontconfig1-dev  # для Ubuntu 24.x

Зависимости (Arch Linux)

alsa-lib
freetype2
libx11
libxcomposite
libxcursor
libxext
libxinerama
libxrandr
libxrender

После установки зависимостей выполните в каталоге с исходниками:

mkdir b
cd b
cmake ..
make
make install

Это установит LV2 и VST3 версии в /usr/local/lib/lv2 и /usr/local/lib/vst3. Чтобы изменить префикс (например, на /usr), используйте:

cmake -DCMAKE_INSTALL_PREFIX=/usr ..
