27 октября 2025 — Drumlabooh 11.1.0

Исправлена повторная загрузка драмкита при "чистом" запуске плагина. Несколько оптимизирован основной цикл работы. Убран интерфейс создания "быстрых китов" — как чрезмерно запутанный, при этом сами "быстрые драмкиты" продолжают поддерживаться на загрузку. Вместо создания "быстрых китов" предлагаю использовать возможность Лабуха по использованию в качестве китов обычных директорий с подпапками с сэмплами.
