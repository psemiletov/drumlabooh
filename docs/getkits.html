As Drumlabooh supports several drum kit formats and has no build-in dedicated manager for kits collection, there are "external" ways to install new drum kits.

Hydrogen drum kits

At start, Drumlabooh scans all installed Hydrogen drum machine kits. There are dozens cool kits are created, but to install them you need to run Hydrogen, and go to Drumkits - Online import menu, where you can download kits from Hydrogen or Musical Artifacts repositories.

Some of Hydrogen kits may be already installed at your Linux. Currently, for Linux and WIndows, the best way to get new Hydrogen kits is Online import feature, because you simple cannot find public download links.

Drumlabooh drum kits

The own kit format of Dramlabooh. One kit = one directory, placed to $HOME/drumlabooh-kits or /usr/share/drumlabooh-kits on Linux, and, for Windows, to c:\drumlabooh-kits or d:\drumlabooh-kits. On its installation, Dramlabooh bundles with kits from Drum Sklad on Github repo. Also, there you can read a details about the format, it is quite simple, you need no more than 5 minutes to create an own kit from the samples.

Additional kits those are not hosted on Github can be downloaded from Drum Sklad at Telegram. So the full Drumlabooh kits collection is Drum Sklad on Github + Drum Sklad on Telegram.

SFZ drum kits

Drumlabooh scans installed SFZ kits at pre-defined locations: $HOME/drumlabooh-kits (Linux) and c:\sfz-kits and d:\sfz-kits (Windows). As usual, SFZ kit directories must be placed there to be available for Drumlabooh.

The kits of excellent quiality - AVL Drumkits - you can download in SFZ format from http://www.bandshed.net/sounds/AVLDrumkits_SF.

Поскольку Драмлабух поддерживает несколько форматов драмкитов, а какого-либо встроенного менеджера коллекций китов у него нет, существуют следующие "внешние" способы установки новых драмкитов.

Драмкиты в формате Hydrogen

При запуске, Drumlabooh сканирует все установленные киты от драм машины Hydrogen , которая выпускается как для Linux, так и Windows. Под Hydrogen существуют десятки драмкитов, но чтобы установить их, надо запустить сам Hydrogen и пойти в нем в меню Драмкиты - Online import, откуда можно скачать киты из двух хранилищ -Hydrogen и Musical Artifacts.

В Линуксе при установке самого Hydrogen скорее всего установятся и некоторые киты. На сегодня, под все системы, самый быстрый способ скачать новые киты для Hydrogen, это его вышеописанная функциияOnline import, ибо публичных ссылок на отдельные киты из хранилищ вы не найдете.

Киты в формате Драмлабуха

Драмкиты в формате Драмлабуха. Один кит = одна папка, помещенная в $HOME/drumlabooh-kits или /usr/share/drumlabooh-kits (в Линукс), либо, для Windows, в c:\drumlabooh-kits или d:\drumlabooh-kits. При установке самого Драмлабуха в комплекте с ним идут киты из хранилища Drum Sklad on Github. Там же вы можете узнать подробности о формате. Дело пяти минут - создание нового драмкита из набора сэмплов.

Дополнительные киты, которых нет на Гитхабе, качайте с Drum Sklad в Телеге. Так что полная коллекция китов для Драмлабуха лежит на Гитхабе ПЛЮС в Телеге.

Драмкиты в формате SFZ

Драмлабух сканирует установленны SFZ-киты $HOME/drumlabooh-kits (под Linux), а также в c:\sfz-kits и c:\sfz-kits (под Windows). Как обычно, папки с SFZ китами надо поместить туда, чтобы Драмлабух их увидел.

Драмкиты превосходного качества - AVL Drumkits - вы можете скачать в формает SFZ по ссылке http://www.bandshed.net/sounds/AVLDrumkits_SF.
