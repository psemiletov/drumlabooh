Возможности

  • Инфа по совместимости с DAW
  • Формат плагинов: Linux LV2/VST3, Windows VST3
  • Поддерживаемые форматы драмкитов: Drumlabooh, Hydrogen, SFZ, обычные папки с сэмплами
  • Драмлабух поставляется с десятками драмкитов, созданными на основе сэмплов легендарных акустических ударных и драм-машин. Звук для любого жанра - от darkwave и psytrance до джаза и рока.
  • Стереовывод (со встроенным микшером), либо 36 отдельных выходных каналов
  • Встроенный микшер с регуляторами панорамы, громкости, заглушения, и выбором режима панорамы
  • Эффекты для каждого инструмента: фильтры низких и высоких частот, а ползунок Analog обеспечит тот самый тёплый ламповый звук.
  • Сэмплерный движок на 36 инструментов с поддержкой 127 слоев в каждом
  • Поддержка множества вариантов сэмплов на один слот
  • Режимы воспроизведения слоев: MIDI velocity, Round Robin, Random Order
  • Умный движок случайных чисел обеспечивает уникальную для песни, но управляемую и постоянную при каждом воспроизведении цепочку значений. Имитация живого звучания конкретной партии всегда будет присуща только ей и постоянной каждый раз, пока вы сами не измените ее параметры.
  • Автоматическое заглушение открытого хай-хета при игре закрытого
  • Определенные в ките (или пользователем) группы заглушения
  • Отображение картинки драмкита, если она включена в состав
  • Удобный и простой MIDI mapping
  • Плагин написан на C++ с использованием тулкита JUCE
