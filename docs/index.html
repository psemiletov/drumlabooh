NEWS

October 10 2023 - 2.0.1 release - Drum kits list initialization fixed.

October 08 2023 - 2.0.0 release
This release introduces the multi-channel (36 outs) version of the plugin. See the comprehensive documentation at the Drumlabooh site.
Also, some Reaper-related things has been fixed.

October 06 2023 - 1.0.0 release
Hello! The first major release ever. All known bugs were fixed, but let's talk about interesting things.
The most cool one is FX section for each instrument. You can call/hide FX with the button "FX". There are two resonance filters (Low pass and High pass) to change the sound, and the magic "Analog" knob that warms up the sample.
Other changes are more GUI-related - palette, drum cells highlighting, etc.
A new "Wooden" drum kit has been added.
Other significant changes:
Documentation updated.
Stereo samples from kits now loaded as mono to simplify internal mixing. Pre-panned drums are evil.

September 07 2023 - 0.0.4 release
* fixed the drumkit loading at Ardour session after opened
* LEDs are velocity-depended now

НОВОСТИ

10 октября 2023 - 2.0.1 release - Исправление инициализации списка драмкитов.

8 октября 2023 - выпуск 2.0.0
Добавлен многоканальный (36 выходов) вариант плагина, подробности об ипсользовании читайте в руководстве на сайте. Также подправлено кое-что для Рипера.

6 октября 2023 - выпуск 1.0.0
Первый крупный релиз! Об исправлениях глюков не будем, давайте про интересное.
Самая классная штука из нового это раздел FX для каждого инструмента. Вызывается кнопкой FX, ею же и прячется. FX открывает окошко с двумя резонансными фильтрами (низкочастотный и высокочастотный), а также волшебной кнопкой "Аналог", которая подогревает звук.
Прочие новинки да изменения больше относятся к графическому интерфейсу, сами увидите.
Также добавлен "Деревянный" драмкит.
Еще важное:
Документация обновлена.
Стерео-сэмплы загружаются как моно, чтобы упросить внутреннее сведение.
