Windows: installation

Download and run this installer – installs to C:\Program Files\Common Files\VST3. It also installs some drumkits to C:\drumlabooh_kits, but you can get more with Hydrogen, or additional Drumlabooh kits manually from Telegram at Drum Sklad (just copy/unpack kit directories to C:\drumlabooh_kits or D:\drumlabooh_kits, or for SFZ format C:\sfz_kits or D:\sfz_kits). Please read the Manual for details.

Linux: Build from the source

To build Drumlabooh from the source, you need to install some dependencies first. Drumlabooh is based on JUCE. The JUCE source code will be fetched during the configuration process, but JUCE requires additional libraries.

Dependencies (Ubuntu packages names):

pkgconf
libasound2-dev
libfreetype6-dev
libx11-dev
libxcomposite-dev
libxcursor-dev
libxext-dev
libxinerama-dev
libxrandr-dev
libxrender-dev
libfontconfig1-dev //for Ubuntu 24.x

Dependencies (Arch packages names):

alsa-lib
freetype2
libx11
libxcomposite
libxcursor
libxext
libxinerama
libxrandr
libxrender

Install them if not installed, and then, under sudo or root:

mkdir b
cd b
cmake ..
make
make install

This installs LV2 and VSTi versions of the plugin to /usr/local/lib/LV2 and /usr/local/lib/VST3 by default. To redefine the destination directory, set another destination via the CMAKE_INSTALL_PREFIX variable. For example, from /usr/local to /usr:

cmake -DCMAKE_INSTALL_PREFIX=/usr ..

Windows: установка

Скачайте и установите этот установщик VST3i, заодно поставится необходимый минимум драм-китов. Затем по вкусу установите дополнительные драмкиты, например в драм-машине Hydrogen, или вручную, формата Drumлабуха с тг-канала Drum sklad. Скопируйте/распакуйте папки драмкитов в C:\drumlabooh_kits или D:\drumlabooh_kits либо, в случае формата SFZ: C:\sfz_kits или D:\sfz_kits. За подробностями обратитесь к Руководству пользователя. Зачитаетесь!

Linux: сборка из исходника

Чтобы собрать Драмлабух из исходника, надо его удовлетворить, то бишь установить зависимости, нужные для библиотеки JUCE, на которой Драмлабух основан и которую подгружает из сети во время сборки.

Зависимости (имена пакетов в Ubuntu):

libasound2-dev
libfreetype6-dev
libx11-dev
libxcomposite-dev
libxcursor-dev
libxext-dev
libxinerama-dev
libxrandr-dev
libxrender-dev
libfontconfig1-dev //для Ubuntu 24.x

Зависимости (имена пакетов для Arch):

alsa-lib
freetype2
libx11
libxcomposite
libxcursor
libxext
libxinerama
libxrandr
libxrender

Установите их, ежели они еще не установлены, а потом выполните из каталога исходника Драмлабуха, под рутом или sudo:

mkdir b
cd b
cmake ..
make
make install

Сие установит Lv2 и VSTi версии плагина в /usr/local/lib/LV2 в /usr/local/lib/VST3. Если вы хотите переопределить каталог назначения, то назначьте его посредством переменной CMAKE_INSTALL_PREFIX. Например, заменить /usr/local на глобальный /usr можно так:

cmake -DCMAKE_INSTALL_PREFIX=/usr ..
