26 декабря 2025 — Drumlabooh 12.1.0

Новый выпуск вызван многочисленными откликами на предыдущий, а также накопившимися предложениями.

Добавлена поддержка MIDI mapping в драмкитах Hydrogen.

Исправлена поддержка MIDI mapping для SFZ.

Связанные со слотами инструментов номера MIDI-нот текущих настроек MIDI Mapping (в режимах Auto и Kit) теперь отображаются как всплывающие подсказки над названиями слотов.

Если опция MIDI map mode = Kit, но в драмките не заданы настройки MIDI mapping, Drumlabooh автоматически использует MIDI map mode = Auto, когда ноты нумеруются последовательно начиная со значения MIDI base note. Было - Drumlabooh просто молчал.

Установленные драмкиты теперь сканируются также в заданных пользователем каталогах, список которых, строка за строкой, в виде полных путей, поместите в файл $HOME/.config/drumlabooh/user-dirs.txt (для Linux) либо в C:\drumlabooh\user-dirs.txt (для Windows). Например: 

/home/test/sfz-main
/home/test/sfz-misc
/home/test/hydrogen-kits
/home/test/drumlabooh

По сути, теперь нет разницы драмкиты какого формата в каком каталоге лежат, Drumlabooh распознает их правильно. Исключение составляют обычные папки с сэмплами - их Drumlabooh тоже может автоматически превращать на лету в драмкиты, но подробности читайте в документации.
